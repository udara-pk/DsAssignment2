Configuration for FIRE ALARM MONITORING SYSTEM
----------------------------------------------------
1) Restore firesensordb.sql file to the database
2) Run SpringBootMysqlApplication.java in spring-boot-mysql project
3) Run FireSensor.java in Firesensor Project
4) Go to web-app and run 'npm install' to install dependancies and then run 'npm start'
5) Run 'start rmiregistry' command
6) Go to Fire-Alarm-Monitoring-System-Desktop project and run ClientApplication.java to launch Desktop app
